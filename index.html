My Projects

FinOps Analytics

Used Python, SQL, and Linux bash commands to build a system that automates the process of assigning cloud usage costs to different departments in the company. This ia a real system that will have a very positive future business impact.
Please click here to read more about what this project entailed.

Munch Application

Website for RPI to allow Students to have a better dining experience. We used beautiful Soup to scrape web for dining hall information. We also worked with dining halls to have menus entered daily, allowed users to track meal swipes as well as actually rate their meals. We used HMTL, CSS, JavaScript Frontend and a Python (Django, Beautiful Soup) Backend.

SwiftHire Application

This application is a project my team and I developed while learning about Software Design and Documentation. Our team decided to use the MERN stack for our technical strategy. We also decided during project planning to adopt the Scrum and Agile methodologies to work together and communicate efficiently.

RCOS: Disbaled Students website

My RCOS team and I created this website for any disabled students that attend RPI. This website has multiple features like text color and size toggles for colorblind people, a campus map with accessible routes, disabled student club discord invite, and a disabled students' club event calendar.

Study of Big O

I decided to do this project because I learned about Big O Notation and that using different data structures can change the efficiency of a process. So I basically took each data structure that I had learned about and tested how long it would take for a task (such as sorting) to be completed.
